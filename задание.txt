Тестовое задание


Задача
Необходимо реализовать сервис со следующим функционалом на языке C# с использованием фреймворка ASP.Net Core 6.0


Требования 


1. Реализовать две консольные команды Producer и Consumer
Producer считывает адреса сайтов из файла. Имя файла передается как аргумент запуска. Адреса сайтов записаны по одному на строку.
Для каждого сайта Producer создает событие содержащее адрес сайта в очереди RabbitMQ
Consumer читает события из очереди RabbitMQ и делает HTTP GET запрос по адресу указанному в событии. Полученный ответ сохраняется в таблицу в базе данных MySQL 8.0 содержащую следующие поля
* id -- PK
* url -- адрес сайта
* date – дата запроса
* response -- тело ответа с сайта


2. Должна быть предусмотрена возможность работы нескольких Consumer одновременно.
